# FastAPI Form() 완벽 정리

## Form()이란?

**HTML 폼에서 보내는 데이터를 받을 때 사용하는 FastAPI 기능**

웹페이지의 로그인 폼, 회원가입 폼 등에서 전송되는 데이터를 서버에서 처리하기 위한 도구입니다.

## 기본 사용법

```python
from fastapi import FastAPI, Form

app = FastAPI()

@app.post("/login")
async def login(username: str = Form(), password: str = Form()):
    return {"username": username, "password": password}
```

## 왜 Form()을 사용하나?

### 데이터 전송 방식의 차이

#### 1. JSON 방식 (일반 API)
```python
@app.post("/user")
async def create_user(name: str, age: int):
    pass
```
- 데이터 형식: `{"name": "철수", "age": 20}`
- Content-Type: `application/json`

#### 2. Form 방식 (HTML 폼)
```python
@app.post("/user")
async def create_user(name: str = Form(), age: int = Form()):
    pass
```
- 데이터 형식: `name=철수&age=20`
- Content-Type: `application/x-www-form-urlencoded`

### HTML 폼은 기본적으로 Form 데이터로 전송

```html
<form method="POST" action="/login">
    <input type="text" name="username">
    <input type="password" name="password">
    <button type="submit">로그인</button>
</form>
```

위 폼 제출 시 → `username=admin&password=1234` 형태로 전송

## Form()을 꼭 써야 하는 경우

### 1. 파일 업로드

```python
from fastapi import File, UploadFile

@app.post("/upload")
async def upload(
    file: UploadFile = File(),
    description: str = Form()
):
    return {"filename": file.filename, "description": description}
```

**JSON으로는 파일을 전송할 수 없음!**
- 파일 + 데이터를 함께 보낼 때는 `multipart/form-data` 필수

### 2. JavaScript 없이 동작하는 웹사이트

- 순수 HTML만으로 폼 제출 가능
- 간단한 사이트에서 유용

### 3. 외부 표준 프로토콜

- OAuth 로그인 등은 Form 데이터 요구
- 외부 시스템 연동 시 형식 맞춰야 함

## JSON vs Form 비교

| 구분 | JSON | Form |
|------|------|------|
| 형식 | `{"key": "value"}` | `key=value&key2=value2` |
| 파일 전송 | ❌ 불가능 | ✅ 가능 |
| JavaScript | 필요 | 불필요 (선택) |
| 사용 예시 | REST API | 로그인 폼, 파일 업로드 |

## 프론트에서 JSON으로 변환 가능?

**가능합니다!** 하지만 상황에 따라 Form이 더 적합한 경우가 있습니다.

### JSON으로 보내는 방법

```javascript
fetch('/login', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({
        username: 'admin',
        password: '1234'
    })
})
```

### Form으로 보내는 방법

```javascript
const formData = new FormData();
formData.append('username', 'admin');
formData.append('password', '1234');
formData.append('profile', fileInput.files[0]); // 파일 추가 가능!

fetch('/login', {
    method: 'POST',
    body: formData  // Content-Type 자동 설정
})
```

## 요약

- **Form()**: HTML 폼 데이터를 받을 때 사용
- **필수 상황**: 파일 업로드, 외부 프로토콜 연동
- **선택 가능**: 단순 데이터 전송 (JSON도 OK)
- **핵심**: 데이터 형식에 맞는 도구를 선택하는 것이 중요!
